# Login Test Application with Cookie Session Management

## Features
- ✅ Secure cookie-based sessions
- ✅ "Remember Me" functionality (30 days)
- ✅ HttpOnly & SameSite protection
- ✅ Persistent and temporary sessions
- ✅ Session information dashboard

## Quick Start
- **Windows**: Run `setup.bat` then `run.bat`
- **Linux/Mac**: Run `./setup.sh` then `./run.sh`
- **Browser**: http://127.0.0.1:5000

## Cookie Features
- **Remember Me**: Check the box to stay logged in for 30 days
- **Security**: HttpOnly cookies prevent XSS attacks
- **Protection**: SameSite attribute prevents CSRF attacks
- **Auto-expiry**: Sessions expire based on configuration
